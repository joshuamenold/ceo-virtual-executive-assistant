        # Obsidian Bases

        Use Obsidian's database-like features for structured data tracking and views.

        ## Overview

        Bases let you create structured views of your notes — tables, boards, and filtered lists — using frontmatter properties as columns.

        ## Usage

        - Create base files that query notes by tags, folders, or properties
        - Define columns from frontmatter fields (status, priority, date, etc.)
        - Use filters and sorts to organize data views
        - Ideal for project tracking, task management, and CRM-style data
